# DrawLab PWA — הוראות פריסה ועדכון

## מה קיבלת?

| קובץ | תפקיד |
|---|---|
| `index.html` | האפליקציה שלך (עם תוספות PWA) |
| `manifest.json` | מגדיר שם, צבע ואייקון לאפליקציה |
| `sw.js` | Service Worker — מאפשר התקנה, עבודה אופליין ועדכונים |
| `icons/icon-192.png` | אייקון לאנדרואיד |
| `icons/icon-512.png` | אייקון גדול (Splash screen) |

---

## שלב 1 — העלאה לאינטרנט (חינמי, 5 דקות)

### אפשרות מומלצת: Netlify Drop

1. גש ל-**[netlify.com/drop](https://app.netlify.com/drop)**
2. גרור את **תיקיית `DrawLab-PWA`** לתוך הדפדפן
3. Netlify יתן לך כתובת כמו `https://amazing-name-123.netlify.app`
4. **זהו!** האתר חי באינטרנט.

### אפשרויות נוספות (חינמיות)
- **GitHub Pages** — אם יש לך חשבון GitHub
- **Vercel** — גרור תיקייה ב-[vercel.com/new](https://vercel.com/new)
- **Cloudflare Pages** — [pages.cloudflare.com](https://pages.cloudflare.com)

## שלב 2 — הוספה למסך הבית באנדרואיד

1. פתח את **Chrome** בטלפון
2. גש לכתובת שקיבלת מ-Netlify
3. תוך שניות יופיע בנר "**הוסף לדף הבית**" — לחץ עליו
4. אם לא הופיע: לחץ על **שלוש הנקודות** ← **"הוסף למסך הבית"**
5. האפליקציה תיפתח ללא סרגל כתובת, בדיוק כמו אפליקציה רגילה

---

## שלב 3 — קבלת עדכונים מ-Claude

כל פעם שתקבל גרסה חדשה של `DrawLab.html` מ-Claude:

1. **שנה את שם הקובץ** ל-`index.html`
2. **עדכן את מספר הגרסה** ב-`sw.js` — שורה 5:
   ```
   const CACHE_NAME = 'drawlab-v1';  ← שנה ל-v2, v3...
   ```
3. **העלה מחדש** את הקבצים ל-Netlify:
   - גרור שוב את התיקייה ל-Netlify Drop, **או**
   - אם השתמשת ב-GitHub: דחוף commit חדש

4. פתח את האפליקציה בטלפון — יופיע **בנר צהוב בתחתית**: "🔄 גרסה חדשה זמינה — לחץ לרענון"
5. לחץ עליו — האפליקציה תתעדכן מיידית

> **חשוב:** עדכון מספר הגרסה ב-`sw.js` הוא ההוראה לטלפון לטעון מחדש את כל הקבצים.
> בלי שינוי הגרסה, הטלפון ימשיך להשתמש בגרסה הישנה מהמטמון.

## שאלות נפוצות

**ש: האם הנתונים שלי נשמרים בין עדכונים?**
כן. הנתונים נשמרים ב-localStorage של הדפדפן במכשיר שלך ואינם נמחקים בעדכון.

**ש: האם זה עובד ללא אינטרנט?**
כן. לאחר הטעינה הראשונה, האפליקציה עובדת גם ללא חיבור (למעט משיכת קבצי CSV חיצוניים).

**ש: מה ה-URL שאני צריך לשמור?**
הכתובת מ-Netlify (למשל `https://drawlab.netlify.app`). תוכל גם לשנות אותה בהגדרות Netlify.
